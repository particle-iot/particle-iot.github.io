Detailed Description

template<size_t MAX_NEIGHBOR_CELLS>
class CellularHelperEnvironmentResponseStatic< MAX_NEIGHBOR_CELLS >

Response class for getting cell tower information with statically defined array of neighbor cells.

Parameters
MAX_NEIGHBOR_CELLStemplated parameter for number of neighbors to allocate. Each one is 44 bytes. Can be 0.

The maximum the modem supports is 30, however you can set it smaller. Beware of using values over 10 or so when storing this object on the stack as you could get a stack overflow.
